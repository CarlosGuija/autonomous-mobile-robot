# Changelog

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Planned

## [0.1.0] - 2026-07-14

### Added

- Configurable velocity publisher
- Odometry-based drive-distance and rotate-angle controllers
- Custom `DriveDistance` and `RotateAngle` ROS 2 actions
- Cancellable motion action servers with feedback and safe-stop handling
- Move-square finite-state machine and combined launch file
- Initial public documentation, contribution guide, and MIT license
